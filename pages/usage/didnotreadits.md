# Creating a didnotreadit

A **didnotreadit** is a community organized around a topic. Anyone with an account can create one.

## Create one

1. Sign in.
2. Go to `/d/create` (or click **create a didnotreadit** in the sidebar).
3. Fill in the form:
   - **name** — the URL slug. 3–50 characters, lowercase letters, numbers, underscores, and hyphens only. This becomes `d/<name>` and cannot be changed later.
   - **title** — a human-readable display title, up to 100 characters.
   - **description** *(optional)* — a sentence or two describing what the community is for. Shown in the didnotreadit's header.
4. Submit.

You're now the creator of the didnotreadit. The creator name is shown on the community page, but creating a didnotreadit doesn't give you any moderation powers — every signed-in user can post and comment in any didnotreadit on the instance.

## Naming tips

- Pick something short and topical. `d/release-notes` is better than `d/general-announcements-and-news`.
- Names are global to the instance — if someone else has already taken `d/foo`, you'll get an error.
- The name is permanent. The title and description can be edited later by the creator; the name cannot.

## Find an existing one

Before creating a new didnotreadit, check `/d/new` to see whether one already exists for the topic.
